# Automated Network Request Management in ServiceNow

📌 Project Overview

This project automates network-related service requests in ServiceNow, improving efficiency and reducing manual work.

⚙️ Features

* Automated request creation
* Approval workflows
* Task assignment
* Status tracking

🛠️ Technologies Used

* ServiceNow
* Workflow Automation
* JavaScript (Server-side scripting)
